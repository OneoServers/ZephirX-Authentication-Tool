Installation Instructions

STEP 1: Download the ZephirX Authentication Tool zip file by clicking the Download Now button

STEP 2: Extract the zip file to a folder on your computer

STEP 3: Open Google Chrome and navigate to chrome://extensions/

STEP 4: Enable "Developer mode" using the toggle in the top right

STEP 5: Click "Load unpacked" and select the extracted folder

STEP 6: The ZephirX extension should now appear in your extensions list

How to Run ZephirX Authentication Tool

STEP 1: Click the ZephirX extension icon in your Chrome toolbar

STEP 2: A popup window will open with the extension interface

STEP 3: Navigate to the Roblox website

STEP 4: While on the Roblox site, click the "Verify Account" button in the ZephirX extension popup

Note: Make sure to extract the zip file before loading it as an unpacked extension. Chrome cannot directly load zip files.
